# hinoko
hinoko is application template for cloudflare workers app with Hono and Bun.

## usage


### package install

```shell
$ bun install
```

### development

```shell
$ bun dev
$ open http://localhost:3000
```

### emulator for workers

```shell
$ bun emulator
```

### deploy

```shell
$ bun deploy
```

### test

```shell
$ bun test
```
